
# Projek Calon B2B

Adalah projek yang mempertemukan kebutuhan bisnis ke bisnis

## Run Locally

Clone the project

```bash
  git clone https://github.com/suryaa6666/projek-calo-b2b
```

Go to the project directory

```bash
  cd projek-calo-b2b
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
